Bando do Chapéu de Palha

O Bando do Chapéu de Palha é um bando de piratas que busca o One Piece, um tesouro que permite ao seu capitão se tornar o Rei dos Piratas.

Tripulantes

  • Luffy - Capitão
  • Sanji - Cozinheiro
  • Zoro - Espadachim
  • Nami - Navegadora
  • Usopp - Atirador
  • Chopper - Médico
  • Robin - Arqueóloga
  • Franky - Mecânico
  • Brook - Músico
